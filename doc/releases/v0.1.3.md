## What changed

- Linux downloads are now statically linked with musl, so LGTM runs on older distributions without requiring the build runner's GLIBC version.
- The public installer now selects the portable Linux artifact automatically.

## Who should upgrade

All Linux users should upgrade from `v0.1.2`. This fixes the `GLIBC_2.39 not found` startup failure. macOS behavior is unchanged.

## Install

```sh
curl -fsSL https://raw.githubusercontent.com/tcbuilds/lgtm/main/scripts/install.sh | sh
```
