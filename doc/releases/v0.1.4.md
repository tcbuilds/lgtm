## What changed

- Added `lgtm update` to securely install the latest public release in place.
- Added `lgtm update --check` for a read-only update check and `--version vX.Y.Z` for pinned updates.
- Updates verify the published SHA-256 checksum and atomically replace the running binary.

## Who should upgrade

All users can upgrade. Policies, schemas, and bundled checks ship inside the binary, so updating LGTM updates those assets too. Repository configuration is still reconciled separately with `lgtm init`.

## Install

```sh
curl -fsSL https://raw.githubusercontent.com/tcbuilds/lgtm/main/scripts/install.sh | sh
```
