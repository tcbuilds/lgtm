## What changed

- Fixed Claude Edit/Write hooks rejecting valid absolute file paths.
- Hook roots now resolve from nested workspaces to the real repository root,
  preventing commands and evidence from leaking across workspace boundaries.
- Fresh init and V1 migration now produce workspace-scoped V2 commands; build
  artifacts are excluded from discovery and nested evidence directories are
  ignored correctly.
- Stop hooks run fast, touched-workspace checks by default. Run
  `lgtm check --tier full` at push/CI boundaries for the complete test, build,
  and coverage gate.
- Added `scripts/lgtm-pre-push`, a reusable local full-gate hook with safe
  installation instructions in the README.

## Upgrade

```sh
lgtm update
```

Repositories using legacy V1 configuration should rerun `lgtm init
--migrate-config` and review the generated workspace commands.
