# v0.2.5

## Fixed

- `lgtm init` no longer re-adds legacy `languages` or `required_commands` keys to a valid V2 config.
- Existing V2 config is preserved byte-for-byte during re-initialization, including user formatting and custom fields.
- V2 configs containing rejected legacy keys now fail before writing, leaving the original file untouched.
